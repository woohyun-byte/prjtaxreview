## 2. 대기업 공제율 구조 (핵심 전제)

> 출처: 조특법 제10조, 시행령 제9조

| 공제 유형 | 대기업 공제율 | 비고 |
|---|---|---|
| 일반 R&D비 (당기분 방식) | 수입금액 대비 비율 × 1/2 (**최대 2%**) | 대기업은 매우 낮음 |
| 일반 R&D비 (증가분 방식) | 25% | 직전 대비 증가분 기준 |
| **신성장·원천기술** R&D비 (별표7) | **20% + α** | α = 매출 대비 R&D비율 × **3배**(시행령 제9조⑤), 최대 10%p |
| **국가전략기술** R&D비 (별표7의2) | **30% + α** | 동일 |

**시사점 (대기업 특화)**
- 일반 R&D 당기분 공제율이 매우 낮으므로, **신성장·원천기술/국가전략기술 분류 여부가 공제액을 좌우하는 핵심 변수**(최대 약 28%p 차이).
- 동일 기술이 별표7·별표7의2 양쪽 해당 시 **공제율 높은 국가전략기술(30%) 선택**.
- 단, 두 공제는 동시 적용 불가(납세자 선택), **구분경리** 전제.

## 3. 전체 추진 프로세스 (7단계)

```
S1 적용가능성 진단   → 연구소 인정여부 / 대상연도(경정청구 5년) 확정
S2 활동·비용 식별    → 별표7·7의2 매핑 + 활동 적격성 평가 + 구분경리
S3 증거서류 구축     → 계획서·보고서·연구노트 (실무부담 최대 단계)
S4 공제액 산정       → 증가분/당기분 시뮬레이션, 최저한세·이월 검토
S5 사전확인 신청     → 기술심의위원회(기술 해당여부) / 국세청 사전심사(비용 적격성)
S6 신고·공제 / 경정청구
S7 사후관리         → 증빙 5년 보관, 세무검증 대응
```

### 매핑 판정 규칙 (필수 적용)
1. **중복 시 국가전략(NS) 우선** — 공제율 높은 쪽 선택
2. **정량요건 충족 검증** — 출력밀도·효율·온도·재기화용량 등 수치기준 충족 여부 확인, 불명확 시 `확인필요`
3. **"조선업 = 자동적용" 아님** — 실제 개발 기술이 조문 정의에 부합하는지로 판정
4. **소부장 정의 확인** — 부품·장비 항목은 「소부장 특별조치법 시행령」 제2조 1·2호 정의 충족 여부 확인
5. **정부지원금 중복배제** — 정부 출연금 보조분은 세액공제 대상 제외, 과제별 정부지원 여부 기록

## 5. 기능요건 B — 연구개발활동 적격성 평가 (7단계 게이트)

> 상세 루브릭은 `연구개발활동_평가기준.md` 참조. 게이트 방식: 앞 단계 탈락 시 이후 평가 무의미.

| STEP | 내용 | 판정 |
|---|---|---|
| 1 | 공통·필수요건 (연구소 신고 / 인력 전담 / 설비 용도) | 미충족 시 **즉시 불인정** |
| 2 | 제외활동 9개 해당 여부 | 해당 시 **불인정** |
| 3 | 5대 판단기준: 신규성·창의성·불확실성·체계성·재현가능성 | 충족/부분/미충족 |
| 4 | 제조/SW 개발단계별 부합도 (✅🔵🟢⚪) | 매핑 |
| 5 | 경계영역(산업디자인·임상·특허 등) 판정 | 평가 |
| 6 | 분쟁사례 대조 | 평가 |
| 7 | 종합판정 + 증빙 보완사항 제시 | 출력 |

### 입력 항목 (과제 입력양식 7항목 + 누락주의 4항목)
- **기본 7항목**: 과제명 / 수행부서 / 참여인력 / 개발배경 / 개발목적 / 개발방법 / 세부목표(n개)
- **누락주의 4항목** (입력 없으면 `확인필요` 처리): ① 인력 전담/겸직 여부 ② 설비 용도(연구용/생산용) ③ 자체개발/위탁 구분 ④ 시제품/양산품 구분

## 6. 데이터·산출물 요구사항

### 6-1. 매핑결과 출력 컬럼 (`매핑결과.csv` / `.md`)
- 과제번호 / 과제명
- 매핑항목ID / 매핑항목명 / 구분(국가전략·신성장·해당없음)
- 적합도(상·중·하)
- 정량요건(충족·미충족·확인필요·해당없음)
- 판정근거(과제 내용이 어느 판정 포인트와 일치하는지 1~2문장)
- 비고(정부지원 여부, 소부장 정의 확인 필요 등)

- 과제 정보 요약 → STEP1~7 결과 → 판정등급(✅🔵🟢⚪) → 증빙 보완 권고

## 8. 기술 해당여부 확인 채널 (2채널)

> 신성장·원천/국가전략기술 해당 여부와 비용 적격성은 아래 두 공식 채널로 사전 확정 가능. 성격이 다르므로 구분 활용.

| 채널 | 근거 | 주체 | 심의 대상 |
|---|---|---|---|
| **연구개발세액공제기술심의위원회** | 시행령 제9조⑮ | 재정경제부·산업통상부 공동 | 연구개발 대상 기술이 **신성장·원천기술/국가전략기술에 해당하는지** 여부 (+ 사업화·연구개발시설 인정) |
| **사전심사 (Pre-screening)** | 시행령 제9조⑰ | 국세청장 | 지출 비용이 **연구·인력개발비에 해당하는지** 여부 |

**활용 전략**: 기술 분류가 쟁점이면 기술심의위원회, 비용 적격성이 쟁점이면 국세청 사전심사. 첫 적용·고액 과제는 두 채널 병행 권장.

1. **경정청구 대상 과거연도** 범위 확정 (법정신고기한 후 5년 이내)
