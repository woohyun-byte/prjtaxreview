## 2. 대기업 공제율 구조 (핵심 전제)

> 출처: 조특법 제10조, 시행령 제9조

| 공제 유형 | 대기업 공제율 | 최저한세 | 비고 |
|---|---|---|---|
| 일반 R&D비 (당기분 방식) | 수입금액 대비 비율 × 1/2 (**최대 2%**) | **적용** | 대기업은 매우 낮음 |
| 일반 R&D비 (증가분 방식) | 25% | **적용** | 직전 대비 증가분 기준 |
| **신성장·원천기술** R&D비 (별표7) | **20% + α** | **제외** | α 산식, 최대 10%p |
| **국가전략기술** R&D비 (별표7의2) | **30% + α** | **제외** | 동일 |

### 2-1. α(추가공제율) 산식 — 시행령 제9조⑤

```
α = (해당군 전사 R&D비 합계 / 매출) × 3,    한도: 10%p
```

- "해당군"은 **신성장·원천기술** 또는 **국가전략기술** 각 군별로 별도 합산.
- 산정 단위는 **회사 전체(전사)** 매출 대비 비율 — 과제별 비율 산정이 아님.
- 일정배수 = **3배** (시행령 제9조⑤).

### 2-2. 최저한세·이월공제 (조특법 제132조·제144조)

- **일반 R&D 세액공제**(당기분·증가분): 대기업 **최저한세 적용대상**(현재 17%) — 한도 초과분은 그 해 공제 불가.
- **신성장·원천기술 / 국가전략기술 세액공제**: **최저한세 적용제외** — 산출세액 한도 내 전액 공제 가능.
- 미공제액(최저한세 한도·산출세액 부족분)은 **10년 이월공제** 가능. 시뮬레이션 시 직전 10년 이월잔액·향후 한도와 함께 검토.

### 2-3. 시사점 (대기업 특화)
- 일반 R&D 당기분 공제율이 매우 낮으므로, **신성장·원천기술/국가전략기술 분류 여부가 공제액을 좌우하는 핵심 변수**(최대 약 28%p 차이).
- 동일 기술이 별표7·별표7의2 양쪽 해당 시 **공제율 높은 국가전략기술(30%) 선택**.
- 단, 두 공제는 동시 적용 불가(납세자 선택), **구분경리** 전제.
- 신성장·국가전략 공제는 최저한세 적용제외이므로 산출세액만 확보되면 전액 활용 가능 — 일반 R&D 대비 실효성 우월.

## 3. 전체 추진 프로세스 (7단계)

```
S1 적용가능성 진단   → 연구소 인정여부 / 대상연도(경정청구 5년) 확정
S2 활동·비용 식별    → 별표7·7의2 매핑 + 활동 적격성 평가 + 구분경리
S3 증거서류 구축     → 계획서·보고서·연구노트 (실무부담 최대 단계)
S4 공제액 산정       → 증가분/당기분 시뮬레이션, 최저한세·이월공제·α 검토
S5 사전확인 신청     → 기술심의위원회(기술 해당여부) / 국세청 사전심사(비용 적격성)
S6 신고·공제 / 경정청구
S7 사후관리         → 증빙 5년 보관, 세무검증 대응
```

## 5. 기능요건 B — 연구개발활동 적격성 평가 (7단계 게이트)

> 상세 루브릭은 `연구개발활동_평가기준.md`(v1.2) 참조. 게이트 방식: 앞 단계 탈락 시 이후 평가 무의미.

| STEP | 내용 | 판정 |
|---|---|---|
| 1 | 공통·필수요건 (연구소 신고 / 인력 전담 / 설비 용도) | 미충족 시 **즉시 불인정** |
| 2 | 제외활동 9개 해당 여부 | 해당 시 **불인정** |
| 3 | 5대 판단기준: 신규성·창의성·불확실성·체계성·재현가능성 | 충족/부분/미충족 |
| 4 | 제조/SW 개발단계별 부합도 (✅🔵🟢⚪) | 매핑 |
| 5 | 경계영역(산업디자인·임상·특허 등) 판정 | 평가 |
| 6 | 분쟁사례 대조 | 평가 |
| 7 | 종합판정 + 증빙 보완사항 제시 | 출력 |

### 입력 항목 (앱 폼 6항목 + 누락주의 4항목)
- **앱 폼 6항목**: 과제번호(선택) / 과제명 / 개발배경 및 목적 / 개발목표(세부목표)(n개) / 주요개발항목 및 개발목표수준 / 개발효과(선택)
- ※ 수행부서·참여인력은 앱 폼에서 수집하지 않음 — 누락 시 `확인필요` 처리
- **누락주의 4항목** (입력 없으면 `확인필요` 처리): ① 인력 전담/겸직 여부 ② 설비 용도(연구용/생산용) ③ 자체개발/위탁 구분 ④ 시제품/양산품 구분

> 실패한 연구개발도 적격(불확실성 충족 입증). 보고서에 실패 사유·중간 데이터 기록 권장.

### 5-2. 호선별 R&D 구분 가이드 (조선업 특례)

> 출처: 「기계금속 가이드라인」 + 조심 2021전4645 + 조심 2017부4566.

조선업은 동일 도면·시방으로 다수 호선을 건조하는 특성상 "**1번선 = R&D 시제품선 / 2번선 이후 = 양산**" 분기 법리가 확립돼 있다.

| 구분 | 인정 방향 | 핵심 입증 요소 |
|---|---|---|
| **1번선 (Lead Vessel)** | 신규성·불확실성 입증 가능한 **시제품선** — 인건비·재료비·시운전비 R&D 인정 여지 | 신규 추진·연료·자동화 시스템 채택 / 선급 신규 승인 / 시운전 데이터 |
| **2번선 이후 (Sister Ship)** | 추가 신규성 입증 부재 시 **양산** — 원칙적 R&D 제외 | 동일 시방 적용, 도면 재사용 시 |
| **2번선 이후 예외** | 핵심 시스템 변경·신규 시험 항목 도입 시 **부분 R&D 인정** | 변경 BOM·변경 시험계획서 별도 입증 |

**관련 분쟁사례 시사점**:
- **조심 2021전4645**: 연구전담부서 외 신차개발 전담인원(생산기술·품질 인원)의 시제품 검사·시험 노무비도 시제품 제작과정의 일부로 인정 → 조선 1번선 PJT팀(설계실 외 인력) 인건비 인정 근거.
- **조심 2017부4566**: 시제품/양산품 재료비를 회계상 명확히 구분 입증한 경우 시제품 재료비만 인정 → **호선별 BOM·자재 분기 회계** 필요.

**Feedback R&D vs 보증수리 (Trouble Shooting)**:
- Feedback R&D(인정 여지): 인도 후 운항 데이터를 **차기 호선·신모델 개선용**으로 분석, 별도 R&D 계획서로 신규 R&D 등록.
- 보증수리·하자대응(불인정): 단순 고장수리·고객응대 — 영업·서비스 활동에 가까움.

## 6. 데이터·산출물 요구사항

### 6-1. 매핑결과 출력 컬럼 (`매핑결과.csv` / `.xlsx`)
- 과제번호 / 과제명
- 매핑항목ID / 매핑항목명 / 구분(국가전략·신성장·해당없음)
- 적합도(상·중·하)
- 정량요건(충족·미충족·확인필요·해당없음)
- 판정근거(과제 내용이 어느 판정 포인트와 일치하는지 1~2문장)
- 비고(정부지원 여부, 소부장 정의 확인 필요 등)

### 6-2. 활동평가 출력 포맷
- 과제 정보 요약 → STEP1~7 결과 → 판정등급(✅🔵🟢⚪) → 증빙 보완 권고
- 세부목표 n개는 목표별 신규성·불확실성·재현가능성 개별 평가 후 과제 전체로 종합

### 6-3. 공제액 시뮬레이션 출력 컬럼 (`공제액_시뮬레이션_결과.csv` / `.xlsx`)
- 시나리오 (보수 / 적극 / 사용자 정의)

## 8. 기술 해당여부 확인 채널 (2채널)

> 신성장·원천/국가전략기술 해당 여부와 비용 적격성은 아래 두 공식 채널로 사전 확정 가능. 성격이 다르므로 구분 활용.

| 채널 | 근거 | 주체 | 심의 대상 | 처리기간 |
|---|---|---|---|---|
| **연구개발세액공제기술심의위원회** | 시행령 제9조⑮ | 재정경제부·산업통상부 공동 | 연구개발 대상 기술이 **신성장·원천기술/국가전략기술에 해당하는지** 여부 (+ 사업화·연구개발시설 인정) | **약 90일**(연장 가능) |
| **사전심사 (Pre-screening)** | 시행령 제9조⑰ | 국세청장 | 지출 비용이 **연구·인력개발비에 해당하는지** 여부 | **약 90일**(연장 가능) |

**활용 전략**:
- 기술 분류가 쟁점 → 기술심의위원회 / 비용 적격성이 쟁점 → 국세청 사전심사. 첫 적용·고액 과제는 두 채널 병행 권장.
- 신고기한(통상 회계연도 종료 후 3개월 말일) 역산하여 **최소 6개월 전** 신청 착수 필요.
- 사전심사·심의위 결과는 신청 시점의 사실관계에 한해 효력. 후속 변경 발생 시 재신청 검토.

## 9. 비용 식별 기준 (별표6)

### 9-1. 비용 산정 단위

- 매핑·활동평가는 **과제 단위**로 수행하나, 신고 시 적격비용은 **회계연도(과세연도) 단위 합산** 구조.
- 흐름: 과제별 매핑·평가 → 과제별 비용 집계 → **연도별 합산** → 신고서 명세서(일반/신성장/국가전략 구분) 작성.
- 다년 과제는 **연도별로 분할 집계** (인건비·재료비·시설비 발생연도 기준).

### 9-2. 비용 구분

| 구분 | 포함 | 제외 |
|---|---|---|
| 자체연구개발 | 연구소·전담부서 연구원 인건비·사회보험료, 재료비, 시설·장비 사용비 | – |
| 위탁·공동연구 | 적격기관 위탁·공동연구비 (§9-3 참조) | ERP·POS 등 사업운영 시스템 개발 위탁비 |
| 공통 제외 | – | 국가·지자체·공공기관 **출연금** 지출분 |

**조선업 비용식별 룰 (선급/감항 인증)** — 「기계금속 가이드라인」 명시:
- ✅ **R&D 인정**: 시제품(1번선) 제작과정에 필수적인 **선급/감항 인증 입회시험비, 시운전비, 검증비** — "항공기 또는 선박을 개발하는 과정에 있어 감항 및 선급 인증은 시제품 제작과정에 필수적인 요소로서 연구개발의 일부에 해당".
- ⚪ **R&D 제외**: 제품판매 단계의 **선급증서 발급 수수료, ISO·CE 등 시장진출용 일반 인증비용** — "고객사 협의 시제품 성능·기술 검인증"은 인정 / "단순 품질·안전인증"은 불인정.
- 호선별 분기: §5-2 호선별 R&D 구분 가이드와 결합해 **1번선 입회시험비 = R&D / 2번선 이후 동일 인증 = 양산** 분기 적용.

### 9-3. 위탁·공동연구 적격기관 (별표6 + 시행규칙 §7조6항)

1. 「고등교육법」상 **대학** (산학협력단 포함)
2. **국공립 연구기관**
3. **정부출연연구기관** (과기정통부·산업부 등 소관)
4. **KOITA 인정 기업부설연구소·연구개발전담부서**
5. **산업기술연구조합**·전문생산기술연구소 등 시행령 지정 기타 기관

**국가전략기술 위탁 — 국내한정 (시행규칙 §7조6항)**:
- ⚠️ 신성장·원천기술 / 국가전략기술 **위탁공동연구개발** 관계 기관은 **국내 소재 기업·기관으로 한정**.
- **해외(국외) 위탁분**은 **국가전략 30%+α 또는 신성장 20%+α 적용 대상에서 제외** → 일반 R&D 또는 불인정으로 강등.
- 글로벌 시험·해외 설계 위탁이 잦은 조선업은 **계약 단계에서 위탁기관 소재지 사전 확인** 필수.

**ERP·POS 등 사업운영 시스템 위탁개발 = R&D 부적격** (조심 2017서4118 참조).

### 9-4. 최신 추가 인정비용 (2026.1.1 이후 발생분) — 시행령 제9조③·⑦, 별표6 개정

- **AI 학습용데이터 구매비** (과기정통부 보도자료, 2026.2.24)

> ⚠️ 별표6 세부 항목번호·적용범위는 시행규칙(제22호, 2026.3.20.) 및 별표6 본문으로 최종 확인할 것.

## 10. 제약 및 확인 필요 항목

### 제약
- 신성장·원천기술 공제와 국가전략기술 공제는 **동시 적용 불가** (납세자 선택) — 조특법 제10조①
- 출연금 지출분은 적격비용 제외 — 시행령 제9조① 단서
- 구분경리(일반·신성장·국가전략 R&D비 별개 회계) 필수 — 시행령 제9조⑫

### 확인 필요 (⚠️)
1. **경정청구 대상 과거연도** 범위 확정 (법정신고기한 후 5년 이내)
2. **이월공제 잔액 점검** — 직전 10년 미공제 잔액 조회 + 본 신고분 한도 검토 (조특법 제144조)
3. **최저한세 영향 분석** — 일반 R&D는 적용대상, 신성장·국가전략은 제외이므로 시뮬레이션에 분리 반영
4. 기업부설연구소/연구개발전담부서 **인정 유효성** 점검 (인정기관: 한국산업기술진흥협회 KOITA)
5. 최종 기술분류·활동평가는 **기술심의위원회 + 국세청 사전심사 + 세무 전문가 검토** 병행 (1차 초안 단정 금지)
6. **후속 개정 모니터링**: 시행령 제36127호 이후 추가 개정·시행규칙 개정 여부를 신고 시점마다 재확인
7. **연계파일 정정 완료**: `연구개발활동_평가기준.md`(v1.2)와 `세액공제_대상기술_조선_참조.md`(v1.2, 미래형운송→별표7의2·수소 선박항목→별표7) 정정 반영됨.
