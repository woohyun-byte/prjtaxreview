# 세액공제 대상기술 — 조선업 참조표 (별표7 / 별표7의2)

> **출처**: 조세특례제한법 시행령 별표7(신성장·원천기술), 별표7의2(국가전략기술) — 대통령령 제36127호, 2026.4.1. 시행
> **용도**: Claude Code 과제 매핑의 기술분류 기준표 (읽기 전용)
> **정정(v1.2)**: 종전 표가 ①「미래형 운송」선박기술을 별표7로, ②수소 선박기술(액화수소 운반선·선박 암모니아 수소생산)을 별표7의2로 분류한 것은 **양쪽이 뒤바뀐 오류**. 법령 원문 대조로 ①→별표7의2, ②→별표7로 정정 완료. 별표7의2 수소 분야엔 **선박 직접 항목이 없음**(수전해·수소차·제철 등).

---

## ⚠️ 핵심 분류 원칙 (반드시 선적용)

- **「미래형 운송 및 이동수단」 = 별표7의2(국가전략, 30%)** — 선박 화물창·운송·추진·자율운항·디지털설계.
- **「수소」·「에너지효율·수송」·「첨단부품」·「탄소중립」·「IT융합」 = 별표7(신성장, 20%)** — 액화수소운반선·암모니아발전·디젤엔진·발전시스템·플랫폼·모터·전자항해 등.
- 한 과제가 양쪽 중복 시 **국가전략(30%) 우선** (법 제10조①, 납세자 선택·구분경리 전제).

## 4. 매핑 판정 규칙 (필수)

1. **「미래형 운송 및 이동수단」은 국가전략(별표7의2)** — 선박 화물창·운송·추진·자율운항·디지털설계는 30%.
2. **「수소」 선박항목(액화수소운반선·암모니아수소생산)은 신성장(별표7)** — 단, 추진·운송시스템에 해당하면 별표7의2 6.아목과 중복 가능.
3. **중복 시 국가전략 우선** — 액화수소 운반선 화물창, 암모니아 추진 등은 30%.
4. **정량요건 검증** — 출력밀도·효율·온도·저감률 등 수치기준, 불명확 시 `확인 필요`.
5. **"조선업=자동적용" 아님** — 실제 기술이 조문 정의에 부합하는지로 판정.
6. **소부장 정의 확인** — 부품·장비 항목은 별표7의2 비고(소부장 특별조치법 시행령 제2조 1·2호) 충족 확인.
7. **정부지원금 중복배제** — 출연금 지출분 제외.

## 5. 사전심사 성공률 제고 전략

1. **법령 문언 밀착** — "가스화물 저장·운송·적하역시스템", "카고핸들링", "ROC", "증발가스 처리" 등 조문 키워드를 과제 목표·계획서에 그대로 반영.
2. **국가전략 분류는 기술심의위원회 사전확인** — 30% 적용 근거 확보(시행령 제9조⑮).
3. **구분경리·증빙 정합성** — 계획서·보고서·연구노트의 기술범위를 별표 항목과 일치(시행령 제9조제13항).
